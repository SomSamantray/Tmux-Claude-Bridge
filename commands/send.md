description: "Send a message to another AI session. Usage: /send <alias> <message>  or  /send all <message>"
---

Send a message to another AI terminal session using AICOMM.

Use the Bash tool to run:
```
aicomm send $ARGUMENTS
```

Show the command output to the user. The response from the target session will be injected back into this conversation automatically once the target finishes processing.
